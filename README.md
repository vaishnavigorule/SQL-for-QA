# SQL for QA

This repository contains SQL concepts and interview-oriented queries that are commonly used by Software Test Engineers and QA professionals.

## Topics Covered

- Basic Queries
- Filtering Data
- Sorting
- Aggregate Functions
- Joins
- Subqueries
- Views
- Constraints
- Practice Questions

## Database Used

MySQL

## Purpose

To strengthen SQL skills required for Manual Testing and Automation Testing interviews.
